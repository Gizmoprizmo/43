
Единое MVP-приложение для производственных файлов
===============================================

Что это:
- Flask-приложение
- единая SQLite-база
- импорт данных из:
  * ДанныеПроизводства.xlsm
  * ПОРТФЕЛЬ ЗАКАЗОВ 3.2.xlsm
  * Планирование производства ПМЗv4.4.xlsm
  * факт_труд.xlsx
  * факт_31032026.XLS
  * склады.XLS

Что уже есть:
- справочники: изделия, техкарты, заводы, заказчики, календарь
- портфель заказов
- план
- факт/труд
- склады
- журнал импорта

Как запустить:
1. Установить Python 3.11+
2. В терминале:
   pip install -r requirements.txt
3. Запустить:
   python app.py
4. Открыть в браузере:
   http://127.0.0.1:5000

Как импортировать:
1. Положить нужные Excel-файлы в одну папку
2. Открыть раздел "Импорт"
3. Вставить путь к папке
4. Нажать "Запустить импорт"

Важно:
- для старых XLS приложение пытается использовать LibreOffice (soffice) для конвертации в xlsx
- если LibreOffice не установлен, лучше заранее пересохранить XLS в XLSX вручную

Это MVP, не полная замена всех старых Excel-макросов.
Но это уже рабочая единая основа, куда можно переносить логику дальше.
